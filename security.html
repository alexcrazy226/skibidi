Vault / Security

Security overview

This page focuses on the practical protections inside the app: not just theory, but the things that matter in real local usage.

Clipboard

Auto cleanup

Copied passwords do not stay in the clipboard forever and are cleared automatically after a delay.

Bruteforce

Login throttling

After several failed login attempts, the app blocks new attempts for a short time.

Secrets

No hardcoded tokens

Telegram secrets are no longer stored inside the source code and are loaded from environment variables.

Error handling

  • silent exception swallowing was reduced in important areas;
  • logging is used for safer debugging;
  • runtime errors are handled more cleanly without spamming the UI.

Input safety

  • avatar loading is safer;
  • broken images should not crash the app;
  • site values are normalized before being stored.
