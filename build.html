Vault / Build

Build and release

This project is not limited to source code. It was also packaged as a working Windows executable, which makes it easier to demonstrate, test, and share.

Build setup

  • PyInstaller was used for packaging;
  • icon support was added for the app window and executable;
  • bundled resources include wallpaper, music, and icon files;
  • the final app can run as a normal desktop program.

Outputs

  • onefile build: Vault.exe;
  • onedir build for easier inspection and release checks;
  • Windows-friendly distribution flow for project presentation.
pyinstaller --onefile --windowed --name Vault main.py

Output:
onefile_dist/Vault.exe
pyi_dist/main/main.exe
